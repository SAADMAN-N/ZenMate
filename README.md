# ZenMate: Your Ultimate Study Companion 🚀

ZenMate is a productivity app designed to help users focus, stay organized, and achieve their study goals with ease. Featuring an intuitive interface and powerful tools, ZenMate provides a seamless and distraction-free study experience. 🌟

---

## Key Features

- **⏳ Pomodoro Timer with Brown Noise**: Stay productive with a customizable timer and soothing background noise.
- **🎵 Spotify Integration**: Access and control your favorite Spotify playlists directly within the app, including curated lofi study tracks.
- **💡 Rest Reminders**: Gentle reminders to hydrate, stretch, and take breaks for peak productivity.
- **🚫 Do Not Disturb Mode**: Minimize distractions while you study.
- **🔮 Future Expansion**: Plans to extend functionality to desktop (via Electron.js) and mobile platforms (via React Native).

---

## Tech Stack

- **🖥 Frontend**: React.js, Tailwind CSS
- **🌐 Backend**: Node.js, Express.js, Firebase
- **🔗 API Integrations**: Spotify Web API, Notification API
- **🎧 Audio Playback**: Howler.js
- **☁️ Deployment**: Vercel (Frontend), Render/Heroku (Backend)
